Ayyeka Frontend Exam

In this exam, we will need to develop a small game named Tic Tac Toe with angular framework.

Game Flow

Start the game by entering two player names and then show the empty board.
Show the current player turn and game state of the board.
At the end of the game show the winner.

Bonus

You will receive extra credit for creative styling and animations

Technical Requirements

Use angular as your primary js framework.
Use any other js libs for styling improvement.
Preserve game state of the board after a browser refresh.
Allow "rollback" one move in the game by pressing the browser back button.
Support responsive layout.
Design the solution so that it can be easily change be the following items:

Board style themes
Support styling themes of the board and the cells so that it can be easily changed in the future - example a dark mode theme.
Board size
Support a configuration parameter to change the board game from 3x3 to larger (10x10) board.
Game winning examination strategy:
Support an easy strategy change of game won examination
